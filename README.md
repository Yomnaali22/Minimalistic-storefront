# Minimalistic storefront

- Category page
- Product page
- Cart page and mini cart

## Technologies:

- React 18.1.0
- Styled-components
- Context Api
- GraphQL

## Setup

To run this project:

1- Clone and run endpoint: https://github.com/scandiweb/junior-react-endpoint

2- npm i

3- npm start
